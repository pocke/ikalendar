Ikalendar

Provide splatoon 2 schedule as iCal format.

Usage

Register the following URL to your calendar app such as Google Calendar. 

https://ikalendar.herokuapp.com/ical/all.ics

Configuration

You can configure the iCal with query strings.

key Description available values default value
title_format Format of event title See format section %{regular:short_rule} / %{gachi:short_rule} / %{league:short_rule}
description_format Format of event description See format section 
%{regular:rule}
* %{regular:map1}
* %{regular:map2}

ガチマッチ: %{gachi:rule}
* %{gachi:map1}
* %{gachi:map2}

リーグマッチ: %{league:rule}
* %{league:map1}
* %{league:map2}
mode Filter by specified mode gachi or league Empty
rule Filter by specified rule clam_blitz, splat_zones, tower_control or rainmaker Empty

Format

The following meta identifiers are available. Replace XXX with regular, gachi or league.
key value example
%{XXX:short_rule} ヤグラ
%{XXX:rule} ガチヤグラ
%{XXX:map1} スメーシーワールド
%{XXX:map1} ムツゴ楼

Example

Calendar for open Zone on gachi match.

raw: https://ikalendar.herokuapp.com/ical/all.ics?mode=gachi&rule=splat_zones&title_format=エリア: %{gachi:map1}, %{gachi:map2}

URI encoded: https://ikalendar.herokuapp.com/ical/all.ics?mode=gachi&rule=splat_zones&title_format=%E3%82%A8%E3%83%AA%E3%82%A2:%20%25%7Bgachi:map1%7D,%20%25%7Bgachi:map2%7D
